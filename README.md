# saltstack-web-javascript-sieve-atkins

## Description
Given a number find all primes.
Salt was used to build the apache server.

## Tech stack
- salt
- jquary 3.5

## Docker stack
- ubuntu:latest

## To run
`sudo ./install.sh -u`
Availble at http://localhost

## To stop
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
